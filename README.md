## ✨ Features

* Dynamic content rendering from JSON
* Simulated API with loading delay
* Interactive carousel with navigation
* Responsive design (mobile, tablet, desktop)
* Skeleton loading animation
* Error handling with retry option
* Clean and modular React architecture

---

## 🧠 Tools & Technologies Used

### ⚛️ React.js

Used as the core frontend library to build reusable UI components. Implemented functional components with hooks like `useState` and `useEffect`.

### ⚡ Vite

Used for fast development and optimized build performance with instant reload and efficient bundling.

### 🎨 HTML5 & CSS3

Used for structuring and styling the application. Implemented modern UI with gradients, spacing, and responsive layouts.

### 🟨 JavaScript (ES6+)

Handled logic such as API simulation, event handling, dynamic rendering, and carousel functionality.

### 🔄 Custom Hooks

Created a reusable hook (`useContent`) for handling API calls, loading states, and error management.

### 🌐 Mock API (Local JSON)

Used `content.json` to simulate backend data. Implemented artificial delay using `setTimeout` to mimic real API behavior.

### 🎠 Carousel Implementation

Built a custom carousel using CSS transform and React state for smooth transitions and navigation.

### ⏳ Skeleton Loading

Implemented shimmer effect using pure CSS to improve user experience during data loading.

### ❌ Error Handling

Handled API failures gracefully with a retry mechanism.

### 📱 Responsive Design

Ensured compatibility across devices using flexible layouts and responsive styling.

### 🔗 Git & GitHub

Used for version control and collaboration.

### 🚀 Deployment (Vercel)

Deployed the project using Vercel for fast and reliable hosting with continuous integration.

## ▶️ Getting Started

### 1. Clone the repository

```
git clone https://github.com/Shanthalad/grafterr-landing-page.git
```

### 2. Install dependencies

```
npm install
```

### 3. Run the development server

```
npm run dev
```
